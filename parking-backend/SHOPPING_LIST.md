# ESP32 Components Shopping List

## Required Components with Links

### Main Components
1. **ESP32 Development Board**
   - ESP32 DevKit V1 or NodeMCU-32S
   - Quantity: 1
   - Price: ~$6-10

2. **RFID RC522 Module**
   - 13.56MHz RFID Reader/Writer
   - Includes 2 cards/tags
   - Quantity: 1
   - Price: ~$3-5

3. **SG90 Servo Motor**
   - 9g Micro Servo (0-180°)
   - For gate control
   - Quantity: 1
   - Price: ~$2-3

4. **IR Obstacle Sensors**
   - 3-wire IR sensor modules
   - Adjustable detection distance
   - Quantity: 8
   - Price: ~$1 each = $8

### Electronic Components
5. **LEDs**
   - 5mm Green LED (1x)
   - 5mm Red LED (1x)
   - Price: ~$0.50

6. **Resistors**
   - 220Ω resistors
   - Quantity: 2
   - Price: ~$0.20

7. **Buzzer**
   - 5V Active Buzzer
   - Quantity: 1
   - Price: ~$1

   - Quantity: 1

9. **Jumper Wires**

   - Female-to-Female (20x)

    - Quantity: 1

## Starter Kits Alternative

Instead of buying individual components, consider:

**ESP32 Starter Kit** (~$30-40)
- Usually includes:
  - ESP32 board
  - Breadboard
  - Jumper wires
  - LEDs, resistors, buzzer
  - Various sensors

**RFID Starter Kit** (~$15-20)
- Includes:
  - RFID RC522 module
  - Multiple RFID cards/tags
  - Connecting cables

## Component Specifications

### ESP32 Board
- Microcontroller: ESP32-WROOM-32
- Flash Memory: 4MB
- WiFi: 802.11 b/g/n
- Bluetooth: v4.2 BR/EDR and BLE
- GPIO Pins: 30+
- Operating Voltage: 3.3V

### RFID RC522
- Operating Frequency: 13.56MHz
- Reading Distance: 0-6cm
- Interface: SPI
- Operating Voltage: 3.3V
- Current: 13-26mA

### IR Sensor
- Detection Range: 2-30cm (adjustable)
- Operating Voltage: 3.3-5V
- Output: Digital (HIGH/LOW)

### SG90 Servo
- Operating Voltage: 4.8-6V
- Torque: 1.8 kg/cm
- Speed: 0.1 sec/60°
- Rotation: 0-180°

## RFID Cards/Tags

The RC522 module usually comes with:
- 2x RFID Cards (credit card size)
- 2x RFID Key Fobs (keychain size)

Additional cards available separately:
- White cards: ~$0.50 each
- Key fobs: ~$0.70 each
- Stickers: ~$0.30 each

## Recommendations

**For Beginners:**
- Buy a complete ESP32 starter kit
- Get pre-wired components when possible
- Start with breadboard prototyping

**For Production:**
- Order extra components (10% spare)
- Use quality connectors
- Consider PCB manufacturing
- Get weatherproof enclosures

**Quality Tips:**
- Choose modules with clear labeling
- Check seller ratings and reviews
- Prefer modules with jumper wires included
- Buy from sellers with good return policy

## Assembly Time

- **Breadboard Prototype:** 2-3 hours
- **Testing & Calibration:** 1-2 hours
- **Permanent Installation:** 4-6 hours
- **Total:** 1-2 days (with breaks)

## Skill Level Required

- **Beginner:** Can assemble with guide
- **No Soldering:** Use breadboard connections
- **Basic Electronics Knowledge:** Helpful but not required
- **Programming:** Code provided, just upload
